# کلاینت اندروید MasterDnsVPN

یک کلاینت اندروید برای اکوسیستم **MasterDnsVPN**.

- **پروژه اصلی (Upstream)**: https://github.com/masterking32/MasterDnsVPN
- **توضیح**: این ریپو شامل اپ اندروید (کلاینت) است که روی هسته‌ی Go پروژه اصلی ساخته شده است.

## دانلود و نصب

1. به بخش **Releases** همین ریپو بروید.
2. آخرین فایل **`universal`** APK را دانلود کنید (پیشنهادی).
3. روی گوشی نصب کنید.

اگر اندروید اجازه نصب نداد، گزینه‌ی **Install unknown apps** را برای مرورگر/فایل‌منیجر فعال کنید.

## شروع سریع (داخل اپ)

برای استفاده، باید یک سرور MasterDnsVPN فعال داشته باشید (دامنه + کلید).

داخل اپ، فیلدهای لازم را وارد کنید:

- **`DOMAINS`**: دامنه تونل (مثال: `v.example.com`)
- **`ENCRYPTION_KEY`**: کلید مشترک از سمت سرور
- **Resolverها**: یک یا چند DNS resolver (به شکل IP یا IP:PORT)

سپس VPN را روشن کنید.

## این اپ چه کاری انجام می‌دهد؟

- یک سرویس VPN اندروید ایجاد می‌کند و ترافیک را از تونل MasterDnsVPN عبور می‌دهد
- بسته به تنظیمات شما می‌تواند حالت پروکسی (SOCKS/TCP) را فراهم کند

## نکات امنیتی

- APK را فقط از **Releases** همین ریپو نصب کنید.
- **کلید رمزگذاری سرور** را محرمانه نگه دارید.
- اگر اسکرین‌شات از تنظیمات می‌فرستید، کلید و هر نوع رمز/اعتبار را مخفی کنید.

## عیب‌یابی

- **آپدیت نصب نمی‌شود / App not installed**: اگر قبلاً نسخه‌ای با امضای متفاوت نصب بوده، آن را حذف کنید و نسخه جدید را نصب کنید.
- **وصل می‌شود ولی اینترنت کار نمی‌کند**: `DOMAINS`، لیست resolverها و کانفیگ سرور را دوباره چک کنید.
- **VPN سریع خاموش می‌شود**: مجوز VPN و محدودیت‌های باتری را بررسی کنید.
